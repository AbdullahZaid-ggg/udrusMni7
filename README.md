# ادرس منيح (UdruS Mni7)

تطبيق ويب لتنظيم الوقت ومتابعة الدراسة، يساعد الطلاب على تتبع جلسات الدراسة وإدارة المواد الدراسية وتحليل التقدم.

## المميزات

- **المواد الدراسية** - إدارة وعرض المواد الدراسية مع المواضيع
- **المؤقت** - مؤقت بومودورو لتنظيم جلسات الدراسة مع ربطها بالمواد
- **التقدم** - متابعة جلسات الدراسة يومياً، أسبوعياً، شهرياً وسنوياً
- **التحليلات** - إحصائيات حية عن ساعات الدراسة والتقدم في المواد
- **الوضع الليلي** - دعم الوضع المظلم

## التقنيات المستخدمة

- React.js
- Dexie.js (IndexedDB) لتخزين البيانات محلياً
- CSS Variables للسمة (الوضع الفاتح/المظلم)

## البدء

```bash
npm install
npm start
```

## البناء للإنتاج

```bash
npm run build
```
